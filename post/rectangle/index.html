Rectangle

 Posted on October 7, 2020  |  2 minutes  |  295 words  |  Jiayu Carl He

Rectangle problems are classic leetcode mono-stack problem and they are one of the hardest problem. That being said, just knowing to use mono-stack to solve the problem is not enough. I am going to dicuss on how to break down those problem for easier understanding and try to teach you a genernal pattern on how to resolve this kind of problem. For sample problems, check Count Submatrices With All Ones , or Largest Rectangle in Histogram

STEP1** Form a recentagle in a histogram. For each bar (denoted as bar and each bar will have its own x value, considering in the x-y cordinate) in a histogram, we can consider if bar can be part of the some rectangle. However, one common characteristics for all those recetanges is that height(recetangle) >= height(bar). And then, we want to think iteratively for every single point in x-axis.

STEP2.1** The largest recetangles that can be formed in this histogram.

Check Largest Rectangle in Histogram for further description. We can start to think about the maximum rectangle that can be formed for every single x1.

STEP2.2** The number of the rectangles in this histogram Check Count Submatrices With All Ones for further description. We want to store the number of submatrices up to i, denoted m[i] and it be used for the future. For example, if height[i+1] >= height[i], that means all submatrices up to i can be extended further to i+1. In addition, we can add new rectangles formed on i+1.

STEP3** 2D -> 1D A histogram is considered as a 1D problem. Sometimes, we have to consider many histograms in a 2D. Therefore, knowing the technique compressing 2D to 1D is extremely helpful. In this cases, we can want to compress rows of histogram and do some calculation.
